# go-expert
Golang repository to study basic and advanced concepts of the language

##

**Package manager**

Initializing:
```
$ go mod init
```

To install packages:
```
$ go get -u github.com/package-name
```

To update version control into go.mod file:
```
$ go mod tidy
```
